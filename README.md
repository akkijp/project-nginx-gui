install
-------

node install

```sh
brew install node
```

electron install

```sh
npm -g install electron-prebuilt
```

nginx-gui start

```sh
git clone https://github.com/k4zzk/nginx-gui.git && cd nginx-gui
npm install
gulp install
gulp run
```
